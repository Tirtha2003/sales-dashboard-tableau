# sales-dashboard-tableau
Interactive Sales Insights Dashboard built using Tableau

## Project Overview
The Sales Insight Dashboard is an interactive business intelligence project developed using Tableau to analyze and visualize sales performance data. The dashboard provides meaningful insights into revenue trends, customer performance, market-wise sales distribution, and overall business growth.

This project helps businesses make data-driven decisions through interactive visualizations and KPI tracking.
